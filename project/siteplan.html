Site Plan

Project: Easy African Recipes

Site Name - Easy African Recipes

This name represents that this site is focused on simplifying recipes for African cuisines.

optional domain: easy-african-recipes.com

Site Purpose

Easy African Recipes is a website that provides a platform for users to learn how to cook African dishes. The website will have a collection of African recipes from different countries in Africa. Users will be able to search for recipes by country, ingredients, and type of dish and get easy steps on making the African cuisine of their choice.

User Scenarios

  • How can I prepare Nigerian Jollof Rice?
  • Are there african salad delicacies for vegetarians?
  • Where can I purchase the local items on my recipe list?
  • What is the spiciest dish in Kenya?

Color Schema

  • Primary Color: #e65100 (Burnt Orange) - Used for headers, buttons, and key highlights.
  • Secondary Color: #ffc107 (Amber) - Used for category tags, section backgrounds, and hover effects.
  • Accent Color: ##1a661c (Dartmouth Green) - Used for success messages, healthy food categories, and decorative accents.
  • Neutral Background: #f9eddc (Antique white) - Used for background to create a warm, inviting feel.

Typography

  • Primary Font: Roboto - for headings and bold texts
  • Secondary Font: Poppins - for paragraphs, links and descriptions
